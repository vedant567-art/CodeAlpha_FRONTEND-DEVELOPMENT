# Simple Calculator

## Project Overview
This project is a basic calculator developed using **HTML, CSS, and JavaScript**. It performs standard arithmetic operations and provides a simple, user-friendly interface for calculations.

## Features

### Core Features
- Addition (+)
- Subtraction (-)
- Multiplication (×)
- Division (÷)
- Clear all input (AC)
- Delete last entered character (DEL)
- Real-time display of user input and results

### User Interface
- Calculator display screen for input and output
- Responsive button layout
- Colorful and interactive design
- Hover effects on buttons

### Keyboard Support
Users can operate the calculator using their keyboard:
- Number keys (0–9)
- Arithmetic operators (+, -, *, /)
- Enter key for calculation
- Backspace key to delete the last character
- Delete or R key to clear the display

## Technologies Used
- **HTML5** – Structure of the calculator
- **CSS3** – Styling and layout
- **JavaScript** – Calculator functionality and keyboard handling

## How to Run

1. Download or clone the project.
2. Ensure both files are in the same folder:
   - `cal.html`
   - `style.css`
3. Open `cal.html` in any modern web browser.
4. Start performing calculations.

## Functionality

### Buttons
- **AC** → Clears the entire display
- **DEL** → Removes the last entered character
- **=** → Evaluates the expression
- **+, -, *, /** → Arithmetic operators

### JavaScript Functions
- `append()` → Adds numbers/operators to the display.
- `deleteLast()` → Removes the last entered character.
- `clearOutput()` → Clears the display.
- `calculate()` → Evaluates the mathematical expression.
